# CHK Help

显示所有 Claude Harness Kit 命令和场景选择指南。

## 可用命令

| 命令 | 描述 | 场景 |
|------|------|------|
| /chk-init | 初始化项目 | 新项目 |
| /chk-solo | 直接对话 | 简单问答 |
| /chk-auto | 全自动修 Bug | 快速修复 |
| /chk-team | 多 Agent 开发 | 日常开发 |
| /chk-ultra | 极限并行 | 大规模重构 |
| /chk-pipeline | 阶段顺序 | 数据库迁移 |
| /chk-ralph | TDD 强制 | 支付/安全代码 |
| /chk-ccg | 三方审查 | 架构决策 |
| /chk-status | 查看状态 | 状态检查 |
| /chk-gc | 知识回收 | 清理知识库 |

## 场景选择指南

- 简单问答 → /chk-solo
- 快速修复 Bug → /chk-auto
- 日常功能开发 → /chk-team
- 批量代码改造 → /chk-ultra
- 数据库迁移 → /chk-pipeline
- 支付/安全关键代码 → /chk-ralph
- 关键架构决策 → /chk-ccg
