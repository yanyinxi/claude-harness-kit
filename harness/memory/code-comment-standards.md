## 规则：所有代码必须写必要的中文注释

**Why:** 项目中发现大量代码没有注释，阅读者难以理解意图，维护成本高。

**How to apply:**
- 函数/方法：必须注释说明 `作用` 和 `参数含义`
- 复杂逻辑：必须注释 `为什么这么写` 而非 `是什么`
- 业务关键点：必须注释 `目的` 和 `预期效果`
- 工具脚本：必须注释 `用途` 和 `使用方式`
- 不写废话注释（如 `// 增加 i` 这种）
- 注释语言：中文优先

**示例:**
```python
def smart_bump(force_type: str = None) -> dict:
    """智能升级版本号

    作用: 根据变更类型自动递增版本号
    原理:
      - patch: 0.0.1 → 0.0.2
      - minor: 0.1.0 → 0.2.0
      - major: 1.0.0 → 2.0.0

    Args:
        force_type: 强制指定版本类型（可选，默认自动分析）
    """
```
